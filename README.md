<!-- ABOUT THE PROJECT -->
## À propos du projet

Ce projet a originé de l’idée que les étudiants du Cégep sont trop peu informés sur la cote r et ne savent pas comment elle est calculée. De plus, même si l’on peut trouver beaucoup d’informations sur internet par rapport à la cote r, nous savons très bien que les étudiants de notre âge sont beaucoup trop paresseux pour faire leur propre recherche sur 10000 sites différents. C’est donc pour cela que notre application est innovatrice. Elle permet à un étudiant paresseux de trouver une multitude d’informations concernant la cote r à un même endroit qui est notre application. Ainsi, si vous voulez plus de détails concernant l’application en tant que telle, veuillez consulter la section<a href="#usage"> Usage </a> de ce README.md

## Usage

*Avertissement*

Il faudrait savoir que cette application ne fonctionne optimalement que sur un téléphone du même size que le pixel 2. Cela va être modifié plus tard afin de pouvoir resize l’application à n’importe quel format de téléphone. Donc, si vous utilisé un émulateur de téléphone pour run l’application, veuillez utilisé un pixel 2 pour une meilleure qualité.

Maintenant, voyons voir ce que l’on peut faire avec l’application. Pour commencer, comme n’importe quelle application, on va pouvoir s’inscrire et se connecter à celle-ci comme on peut le voir ci-dessous :
